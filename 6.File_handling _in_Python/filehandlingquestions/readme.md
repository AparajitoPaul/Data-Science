# File Handling in Python

This folder contains practice questions and exercises related to **File Handling in Python**.

## Important Note

Since many of these exercises involve creating, reading, writing, copying, appending, and modifying files, you will notice a large number of files in this directory.

Some files are:

- **Original sample files** provided for testing.
- **Generated files** created while solving file handling exercises.
- **Output files** produced by Python programs.
- **Copied or merged files** created as part of practice questions.

Because of this, it may not always be obvious which files were manually created and which were generated during execution.

## Using Your Own Files

Feel free to replace the sample files with your own documents for testing. Most programs in this section work with any text, JSON, CSV, or binary file, provided that the filename in the code is updated accordingly.

## Sample Files Included

Examples of files used throughout the exercises:

- `sample.txt`
- `data.csv`
- `data.json`
- `sample.bin`
- `example.txt`
- `example.bin`

Additional files such as `output.txt`, `merged.txt`, `copy_sample.bin`, etc., may be automatically generated when running the notebooks or scripts.

## Recommendation

To better understand file handling concepts, try:

1. Creating your own sample files.
2. Modifying file contents.
3. Running the programs multiple times.
4. Observing how new files are created and existing files are modified.

This hands-on experimentation will help you gain a stronger understanding of Python file operations.
